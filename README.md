# 📊 E-Commerce Business Sales Analytics Platform

A full-stack data-driven analytics platform that helps e-commerce businesses track, analyze, and optimize their sales performance through interactive dashboards and intelligent insights.

---

## 🚀 Overview

This project transforms raw sales data into meaningful business insights. It enables users to visualize trends, monitor key performance indicators, and make informed decisions using an intuitive interface.

---

## ✨ Features

- 📈 **Sales Dashboard**
  - Daily, monthly, and yearly sales trends
  - Revenue visualization using charts

- 🛍️ **Product Performance Analysis**
  - Top-selling and low-performing products
  - Category-wise breakdown

- 👥 **Customer Insights**
  - Purchase frequency tracking
  - Customer segmentation (new vs returning)

- 📊 **Advanced Analytics**
  - Basic revenue forecasting
  - Trend analysis

- 🔍 **Search & Filters**
  - Filter by date, category, product
  - Dynamic queries

- ⚡ **Responsive UI**
  - Clean and simple interface

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/ecommerce-sales-analytics.git
cd ecommerce-sales-analytics
```

### 2. Install dependencies
```bash
cd server
npm install

cd ../client
npm install
```

### 3. Run the project
```bash
# Start backend
cd server
npm start

# Start frontend
cd client
npm start
```
